# Chaney Hunt | UI/UX Portfolio

### Project Description
A personal portfolio built with React.js and Vite to showcase my work as a Junior Digital Media student at the University of Central Florida. This project highlights my specialization in UI/UX design and front-end development, focusing on the intersection of storytelling and functional code.

### Features
- **Single Page Application (SPA)**: Seamless navigation using internal linking.
- **Project Showcase**: Detailed case studies for "Critter & Cauldron," "The Girly Pops," and "Done."
- **Responsive Design**: Optimized for desktop and mobile viewing.
- **Interactive Contact Form**: A user-friendly section for professional inquiries.

### Tech Stack
- **Framework**: React (Vite)
- **Styling**: Vanilla CSS
- **Design Tools**: Figma, Canva, Procreate

### Setup Instructions
1. Clone the repository: `git clone https://github.com/chaneyhunt/my-portfolio.git`
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
